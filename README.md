# 软件上传系统

一个完整的PC端软件下载站第三方用户自行上传软件系统，支持用户注册、登录、软件上传和管理员审核等功能。

## 技术栈

### 前端框架
- **Vue 3** - 渐进式JavaScript框架
- **Vite** - 下一代前端构建工具
- **Vue Router** - 官方路由管理器
- **Pinia** - Vue 状态管理库

### UI组件库
- **Element Plus** - 基于Vue 3的组件库
- **@element-plus/icons-vue** - Element Plus图标库

### 辅助工具
- **Axios** - HTTP请求库
- **Day.js** - 轻量级日期处理库

## 功能模块

### 用户端功能
1. **用户认证**
   - 用户注册（邮箱验证）
   - 用户登录
   - 退出登录

2. **软件上传**
   - 软件基本信息填写（名称、版本、分类、简介等）
   - 软件文件上传（支持最大500MB）
   - 软件截图上传（最多5张）
   - 上传进度显示

3. **个人中心**
   - 个人信息查看和编辑
   - 上传统计数据
   - 查看上传历史
   - 审核状态跟踪

4. **我的上传**
   - 查看所有上传的软件
   - 查看审核状态
   - 查看审核意见
   - 删除待审核软件

### 管理端功能
1. **管理员登录**
   - 独立的管理员登录页面
   - 测试账号：admin / admin123

2. **审核管理**
   - 待审核软件列表
   - 软件详情查看
   - 审核操作（通过/拒绝）
   - 审核意见反馈

3. **软件管理**
   - 查看所有软件
   - 按状态筛选
   - 删除软件

4. **用户管理**
   - 查看所有注册用户
   - 查看用户上传统计

## 快速开始

### 安装依赖

```bash
cd frontend
npm install
```

### 启动开发服务器

```bash
npm run dev
```

项目将在 http://localhost:5173/ 启动

### 构建生产版本

```bash
npm run build
```

## 使用说明

### 普通用户

1. **注册账号**
   - 访问系统后会自动跳转到登录页
   - 点击"立即注册"进行注册
   - 填写用户名、邮箱和密码

2. **登录系统**
   - 使用注册的用户名和密码登录
   - 登录后自动跳转到上传页面

3. **上传软件**
   - 填写软件基本信息
   - 选择软件文件（不超过500MB）
   - 可选择上传软件截图（最多5张）
   - 提交后等待管理员审核

4. **查看上传记录**
   - 在"我的上传"页面查看所有上传的软件
   - 查看审核状态（待审核/已通过/已拒绝）
   - 查看审核意见

### 管理员

1. **登录后台**
   - 点击用户登录页的"管理员登录"链接
   - 使用管理员账号登录
   - 默认账号：admin / admin123

2. **审核软件**
   - 在审核管理页面查看所有待审核软件
   - 点击"立即审核"查看软件详情
   - 选择通过或拒绝，可填写审核意见
   - 提交审核结果

3. **管理软件**
   - 在软件管理页面查看所有软件
   - 可按状态筛选
   - 可删除软件

4. **管理用户**
   - 在用户管理页面查看所有注册用户
   - 查看用户上传统计

## 数据存储

由于本项目为前端演示项目，暂未连接后端服务器，所有数据均存储在浏览器的 localStorage 中：

- `token` - 用户登录令牌
- `userInfo` - 用户信息
- `isAdmin` - 是否为管理员
- `registeredUsers` - 注册用户列表
- `softwares` - 软件列表

**注意：** 清除浏览器缓存将导致所有数据丢失。

## 路由说明

### 用户端路由
- `/login` - 用户登录
- `/register` - 用户注册
- `/user/profile` - 个人中心
- `/user/upload` - 软件上传
- `/user/my-uploads` - 我的上传

### 管理端路由
- `/admin/login` - 管理员登录
- `/admin/review` - 审核列表
- `/admin/review/:id` - 审核详情
- `/admin/users` - 用户管理
- `/admin/softwares` - 软件管理
